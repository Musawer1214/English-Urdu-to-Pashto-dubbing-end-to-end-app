# Roadmap

## Future Features

- **Enhanced Search Functionality**: Improve the search capabilities to allow users to find content more easily.
- **User Profiles**: Allow users to create profiles where they can save their preferences and history.
- **Collaborative Editing**: Enable multiple users to work on dubbing projects simultaneously.
- **Mobile Application**: Develop a mobile version of the app for iOS and Android.
- **Offline Mode**: Implement functionality for users to download and use the app without an internet connection.

## Milestones

- **Q1 2026**: Launch beta testing for enhanced search functionality.
- **Q2 2026**: Release of user profiles feature.
- **Q3 2026**: Start development of mobile applications.
- **Q4 2026**: Complete collaborative editing feature.
